# Project 13: Verifiable On-Chain Job Board

## Vision
A job board on Base where every skill is Merit-attested and candidate profiles anchored to Talent builder score, enabling hiring managers to verify credentials trustlessly.

## Partners
- **Talent Protocol**: Provides builder scores and on-chain profiles
- **Merit Systems**: Skill attestation and verification
- **Base**: Ethereum L2 for deployment

## Core Features
1. **Job Listings**: On-chain job postings with clear requirements
2. **Candidate Profiles**: Linked to Talent Protocol builder scores
3. **Skill Attestation**: Every claimed skill is Merit-attested
4. **Trustless Verification**: Hiring managers can verify credentials directly on-chain
5. **Application System**: Candidates apply with verified credentials

## Technical Stack
- Frontend: Next.js + TypeScript + Tailwind
- Backend: Node.js + Express (or Next.js API routes)
- Smart Contracts: Solidity on Base
- APIs: Talent Protocol API, Merit Systems API
- Storage: IPFS/Pinata for resume/document storage
