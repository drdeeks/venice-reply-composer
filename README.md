## 🎯 Problem

Social media users in crypto face three friction points simultaneously:

1. **Privacy leakage** — asking ChatGPT "what do you think about $DEGEN?" reveals your alpha to centralized AI providers who store and train on your queries
2. **Context switching** — see token mentioned → open new tab → find DEX → swap → come back. Flow broken, momentum lost
3. **No identity verification** — bots and agents post without accountability; humans can't distinguish autonomous agents from spam

This extension solves all three from a single browser extension.

---

## 💡 Solution

Venice AI Reply Composer injects AI-powered reply buttons directly into social media posts. Click once to get 5 context-aware reply suggestions generated by **Venice AI** (no data retention, private inference). See a token you want to trade? Hit the swap button — everything happens in-extension, no tab switching, no redirect.

### THREE-TIER AI FAILOVER CHAIN

```
Venice AI (primary)          → No data retention, uncensored, private
    ↓ (if unavailable)
Bankr LLM Gateway (fallback) → 20+ models: Claude, GPT, Gemini, Llama
    ↓ (if unavailable)
GitHub Models (free tier)    → GPT-4o-mini, Phi-4, Llama-405B
```

No single point of failure. If Venice goes down, replies keep working.

## ⚙️ Configuration

Open the extension popup → Settings tab.

### Venice AI (Required for private inference)
```
API Key: VENICE_INFERENCE_KEY_xxxx...
Model:   venice-uncensored (default) | llama-3.3-70b | mistral-nemo | deepseek-r1
```
Get key: https://venice.ai → API → Create Key

### Bankr (Optional — enables LLM fallback + swaps)
```
API Key: your-bankr-api-key
Model:   gemini-2.5-flash | claude-sonnet-4-5 | gpt-4o
```
Get key: https://bankr.bot → API

### GitHub Models (Optional — free fallback)
```
Token: ghp_xxxx...
Model: gpt-4o-mini | gpt-4o | Phi-4 | Meta-Llama-3.1-70B
```
Get token: https://github.com/settings/tokens → Models permission

### Response Types
Toggle which reply styles to generate:
- ✅ **Agree (Reply)** — supportive, agreeable response
- ✅ **Against (Reply)** — counterpoint, critical perspective
- ☐ **For (Quote)** — quote tweet agreeing
- ☐ **Against (Quote)** — quote tweet disagreeing

## 📖 Usage

### Reply Suggestions
1. Visit Farcaster, Twitter, or Reddit
2. Find a post — look for the **[✨ Reply]** button below it
3. Click → 5 AI-generated suggestions appear
4. Click any suggestion to insert it into the reply box

### Token Swaps
1. When a post mentions a token (e.g. `$DEGEN`), a **[🔄 Swap]** button appears
2. Click → in-extension trade modal opens
3. Connect wallet (MetaMask or any injected wallet)
4. Enter amount in ETH or USD (toggle between modes)
5. Click **Swap via Bankr** — executes via Bankr Agent API
6. If Bankr unavailable → falls back to Uniswap V3 on Base mainnet

### Manual Mode (Farcaster)
If no posts are detected, a **Venice Manual Mode** panel appears bottom-right. Click **Generate Suggestions** → paste any text → get AI replies.
