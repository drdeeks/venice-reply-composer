# Project 8: Give Your AI Agent an ENS Identity

## Partners
- **Virtuals Protocol** (https://virtuals.io)
- **ENS** (Ethereum Name Service)

## Overview
A registration tool that gives any Virtuals AI agent its own .eth ENS name and complete on-chain identity profile. The tool automates:
- ENS name registration/transfer
- Agent profile creation with description and metadata
- Text record storage for capabilities
- Payment address configuration

## On-Chain Footprint
- ENS Registry (Mainnet)
- ENS Resolver (default or custom)
- ERC-721 ENS name NFT transfer
- Text records (key-value metadata)
- Address records for ETH and token payments
